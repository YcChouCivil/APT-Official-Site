關於 APT

1. 核心原則：照片檔案本地化處理

APT 是一款專為處理鑑定照片設計的工具軟體。我們深知案件資料的敏感性，因此在核心功能上採用 「本地處理 (Local Processing)」 技術：

  • 照片與文件不外流： 您所選擇的照片、資料夾路徑及產出的 Word/PDF 報告，僅在您的個人電腦記憶體與硬碟中處理，絕不會上傳到我們的伺服器或任何第三方雲端空間。
  • 無內容存取權： 開發者無法查看、存取或備份您的任何案件照片或文件內容。

2. 雲端服務與資料收集 (PlayFab)

為了提供「點數儲值」與「會員權限管理」功能，本軟體使用 Microsoft Azure PlayFab 後端服務。我們會收集並傳輸以下資訊至雲端伺服器：

  • 裝置識別資訊： 為了識別您的權益，我們會讀取您的電腦名稱 (Machine Name) 與使用者帳戶名稱 (User Name) 建立唯一的識別 ID。請注意：此 ID 綁定於特定裝置，若更換電腦可能導致點數無法直接轉移，需聯繫客服處理。
  • 虛擬商品紀錄： 您的點數餘額與會員訂閱狀態將儲存在 PlayFab 雲端資料庫中，以防止因本地電腦故障或重灌而導致資料遺失。
  • 交易紀錄： 當您透過 Microsoft Store 購買點數時，我們會將交易完成的狀態與金額紀錄傳送至 PlayFab 進行雲端存檔，以確保入帳正確。

3. 資訊安全

您與 PlayFab 伺服器之間的所有連線均採用業界標準的 SSL/HTTPS 加密傳輸。針對 Microsoft Store 的金流支付，我們不會接觸或儲存您的信用卡號。
